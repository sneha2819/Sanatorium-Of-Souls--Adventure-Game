ABSTRACT:

The project aims to showcase the effective use of Java as a programming language, 
with a focus on data structures like Binary Trees and ArrayListsto create an 
adventure game that is both entertaining and challenging.

“Sanatorium of Souls” demonstrates the versatility and adaptability of these 
data structures in game development, providing players with a
unique and dynamic gaming experience.

MODULE-WISE DESCRIPTION:

1. Game-Engine Module:
Utilizes ArrayLists to keep track of the player's inventory, character attributes 
and items collected.

2. Challenge Module:
● Designs and implements various puzzles, challenges and obstacles for players to overcome.
● Utilizes binary trees to structure puzzle solutions and narrative consequences 
based on player choices.
